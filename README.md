# TagFilter - LoRA Dataset Tag Editor

WD14 Tagger を使って画像に自動でタグを付け、LoRA学習用のデータセットを効率よく作成・編集するツールです。

## 概要

LoRA学習では、学習画像1枚につき同名の `.txt` ファイルにタグ（カンマ区切りのキーワード）を記述する必要があります。  
このツールは以下の作業を自動化・効率化します。

- **自動タグ付け** — WD14 Tagger（ONNX形式）で画像を解析し、タグを自動生成
- **タグの確認・編集** — 生成されたタグを画像と並べて確認しながら編集
- **カテゴリフィルタ** — 顔・体・服装・ポーズ・背景・スタイルでタグを絞り込み表示
- **LoRAモード別推論** — 顔LoRA・服装LoRAなど、目的に合わせたカテゴリのみ抽出
- **一括操作** — 全画像へのタグ追加・削除・カテゴリごとの一括削除

---

## 動作環境

| 項目 | 要件 |
|---|---|
| OS | Windows 10 / 11 |
| .NET | .NET Framework 4.8.1（Windowsに標準搭載） |
| GPU | DirectX 12 対応GPU推奨（NVIDIA / AMD / Intel 全対応） |
| メモリ | 8GB以上推奨（大型モデル使用時は16GB以上） |
| ストレージ | モデルファイル用に最大5GB程度の空き容量 |

> GPUがなくてもCPUで動作します。処理速度はGPUの方が大幅に速いです。

## ダウンロードとインストール

1. [Releases](https://github.com/unaya-git/TagFilter/releases/latest) ページから最新版の `TagFilter_vX.X.X.zip` をダウンロード
2. ZIPを任意のフォルダに展開（例：`C:\Tools\TagFilter\`）
3. `TagFilter.exe` をダブルクリックして起動

インストール作業は不要です。レジストリへの書き込みも行いません。  
アンインストールはフォルダごと削除するだけです。

## 初回起動

初回起動時、またはモデルファイルが存在しない場合は以下のダイアログが表示されます。

```
モデルファイルが見つかりません。
Hugging Face から自動ダウンロードしますか？（約365MB）
```

**「はい」** をクリックするとモデルファイルのダウンロードが始まります。  
ダウンロード中はプログレスバーで進捗が確認できます。

> モデルファイルはexeと同じフォルダに保存されます。  
> 複数のモデルをダウンロードした場合もすべて同じフォルダに保存されます。

## 基本的な使い方

### 1. 画像フォルダを開く

「**フォルダを開く**」ボタンをクリックし、LoRA学習用の画像が入ったフォルダを選択します。

対応画像形式：`.jpg` `.jpeg` `.png` `.webp`

フォルダを開くと左側の一覧に画像のサムネイルが表示されます。  
すでに同名の `.txt` ファイルがある場合は、そのタグも自動的に読み込まれます。

> **OneDriveをお使いの場合：** オンライン専用（ローカルに実体がない）ファイルは読み込めません。  
> 対象フォルダ内のファイルを右クリック→「**常にこのデバイスに保存**」を実行してから開いてください。

---

### 2. 自動タグ付けを実行する

「**自動タグ付け**」ボタンをクリックすると、読み込んだ全画像に対してAIによるタグ付けが実行されます。

完了すると各画像の `.txt` ファイルが自動保存されます。

#### 対象画像を絞る

左側の一覧で複数の画像を選択（Ctrl+クリック / Shift+クリック）してから「自動タグ付け」を実行すると、**選択した画像のみ**にタグ付けが行われます。  
何も選択しない場合は**全画像**が対象になります。

## 自動タグ付けの詳細設定

### モデルの選択

ツールバーの「**モデル:**」ComboBoxで使用するWD14モデルを選択できます。

| モデル名 | サイズ | 特徴 |
|---|---|---|
| WD ViT v2（標準・軽量） | 約365MB | 標準的なアニメ・イラスト向けモデル |
| WD SwinV2 v2（高精度） | 約365MB | v2系の高精度版 |
| WD ViT v3（実写向け改善） | 約365MB | v3。実写写真にも比較的対応 |
| WD ViT Large v3（高精度） | 約1.2GB | v3大型モデル。精度高・処理低速 |
| WD Eva02 Large v3（最高精度） | 約2.5GB | v3最大モデル。最高精度・最低速 |

- 初回選択時にモデルが未ダウンロードの場合、自動でダウンロードが始まります
- ダウンロード済みのモデルはインジケータに「**DL済**」、現在使用中は「**使用中**」と表示されます
- 実写写真のタグ付けには **v3シリーズ** が適しています

### 推論カテゴリ（LoRAモード）

「**推論:**」ComboBoxで、タグ付け結果をどのカテゴリに絞るか選択できます。

| 設定 | 抽出されるタグ |
|---|---|
| 全タグ（絞り込みなし） | 全カテゴリのタグ |
| 顔 LoRA | 顔・表情・目・髪型に関するタグのみ |
| 服装 LoRA | 衣服・アクセサリーに関するタグのみ |
| 体 LoRA | 体型・体の部位に関するタグのみ |
| ポーズ LoRA | 姿勢・ポーズに関するタグのみ |
| 背景 LoRA | 背景・シチュエーションに関するタグのみ |
| スタイル LoRA | 画風・スタイルに関するタグのみ |

特定の特徴（顔や服装など）だけを学習させたい場合に使うと、不要なタグが混入しにくくなります。

### 閾値（スコアのしきい値）

スライダーで `0.10` ～ `0.90` の範囲で設定できます（デフォルト：`0.35`）。

- **低くする** → より多くのタグが検出される（精度は下がる）
- **高くする** → 確信度の高いタグのみ検出される（タグ数は減る）

### デバイス

| 設定 | 動作 |
|---|---|
| 自動 | GPU(DirectML)が使えれば自動でGPUを使用、なければCPU |
| GPU | 強制的にGPU(DirectML)を使用 |
| CPU | 強制的にCPUを使用 |

**DirectML** はWindows標準のDirectX 12ベースの技術で、NVIDIA・AMD・Intel すべてのGPUで動作します。追加ドライバのインストールは不要です。

### 並列処理数

`-` / `+` ボタンまたは直接入力で設定できます。

- **GPU使用時**：`1` を推奨（並列化してもVRAMの制約でほとんど速くなりません）
- **CPU使用時**：CPUコア数の半分程度が目安（デフォルト：コア数 ÷ 2）

## タグの編集

### 個別タグの削除

各タグの右にある **`[x]`** をクリックすると、その1件のタグが削除されます。  
削除は即時反映され、`.txt` ファイルも自動保存されます。

### 選択した画像にタグを追加

左側の一覧で画像を選択した状態で、画面下部の「**選択画像にタグを追加:**」欄にタグ名を入力して「追加」ボタンを押すか、Enterキーを押します。

- スペースは自動的にアンダースコア（`_`）に変換されます
- すでに同じタグが存在する場合は追加されません

### 一括タグ挿入

「**一括挿入（全画像）:**」欄にタグ名を入力して「挿入」ボタンを押すと、全画像（または選択中の画像）に同じタグを一括追加できます。

> 例：全画像に `masterpiece` や `best_quality` を追加したい場合に便利です。

---

## 不要タグの削除

### 特定のタグを一括削除

「**不要タグ削除（全画像）:**」欄にタグ名を入力して「**このタグを全削除**」ボタンを押すと、全画像（または選択中の画像）からそのタグを削除できます。

削除前に対象件数を表示する確認ダイアログが出ます。

### 不要タグを登録する

よく削除するタグは「**登録**」ボタンで不要タグとして保存できます。  
登録したタグはチップ（ボタン）として表示され、クリックするだけで削除対象にセットされます。

```
[solo ×][blue_skin ×][white_skin ×][1girl ×]
  ↑クリックで削除対象にセット  ×で登録から削除
```

- `×` をクリックするとそのタグを不要タグ一覧から削除できます
- 登録した不要タグは設定ファイルに保存され、次回起動時も引き継がれます

### カテゴリごとの一括削除

表示フィルタで特定カテゴリ（例：「顔」）を選択すると、右上の「**「顔」タグを全削除**」ボタンが有効になります。  
クリックすると確認ダイアログの後、全画像からそのカテゴリのタグをまとめて削除します。

> 例：服装LoRAを作る際に、不要な顔タグを全削除する場合などに使います。

## 表示フィルタ

右ペイン上部のボタンでタグの表示をカテゴリで絞り込めます。

| ボタン | 表示されるタグ |
|---|---|
| すべて | 全タグを表示 |
| 顔 | 顔・表情・目・眉・口・髪型など |
| 体 | 体型・胸・腰・脚など |
| 服装 | 衣服・靴・帽子・アクセサリーなど |
| ポーズ | 姿勢・ポーズ・動作など |
| 背景 | 背景・環境・シチュエーションなど |
| スタイル | 画風・色調・アートスタイルなど |
| その他 | 上記に分類されないタグ |

フィルタは表示の絞り込みのみで、タグ自体を変更・削除するものではありません。

---

## 設定の保存

以下の設定は終了時に自動保存され、次回起動時に復元されます。

- 選択中のモデル
- デバイス設定（自動/GPU/CPU）
- 並列処理数
- 推論カテゴリ（LoRAモード）
- 閾値
- 登録済み不要タグ一覧

設定ファイルは `TagFilter.exe` と同じフォルダの `settings.xml` に保存されます。  
設定をリセットしたい場合は `settings.xml` を削除してください。

## トラブルシューティング

### 起動しない・すぐ落ちる

**.NET Framework 4.8.1** がインストールされているか確認してください。  
Windows 11には標準搭載されています。Windows 10の場合は [Microsoftの公式サイト](https://dotnet.microsoft.com/download/dotnet-framework) からインストールできます。

### サムネイルが表示されない（OneDriveエラー）

OneDriveのオンライン専用ファイルは読み込めません。  
対象フォルダのファイルを選択 → 右クリック → 「**常にこのデバイスに保存**」を実行してください。

### 自動タグ付けが動かない（2回目以降）

「自動タグ付け」ボタンを押した際にデバイス設定が前回から変わっている場合、モデルの再ロードが行われます。  
エラーメッセージが表示された場合はその内容をご確認ください。

### タグが全く検出されない・少なすぎる

- 閾値が高すぎる可能性があります → スライダーを `0.20` ～ `0.30` 程度に下げてみてください
- 実写写真の場合は **WD ViT v3** 以降のモデルに切り替えると改善することがあります

### `blue_skin` など肌色タグが誤検出される

WD14 Taggerはアニメ・イラスト向けモデルのため、実写画像の肌色を誤認識することがあります。  
不要タグとして `blue_skin` `white_skin` `orange_skin` などを登録しておき、タグ付け後に一括削除してください。

## 利用モデルについて

本ツールは [SmilingWolf](https://huggingface.co/SmilingWolf) 氏が公開している WD14 Tagger モデルを使用しています。

| モデル | Hugging Face リポジトリ |
|---|---|
| WD ViT v2 | SmilingWolf/wd-v1-4-vit-tagger-v2 |
| WD SwinV2 v2 | SmilingWolf/wd-v1-4-swinv2-tagger-v2 |
| WD ViT v3 | SmilingWolf/wd-vit-tagger-v3 |
| WD ViT Large v3 | SmilingWolf/wd-vit-large-tagger-v3 |
| WD Eva02 Large v3 | SmilingWolf/wd-eva02-large-tagger-v3 |

モデルファイルは初回使用時にHugging Faceから自動ダウンロードされます。
